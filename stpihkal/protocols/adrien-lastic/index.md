# Adrien Lastic

## Overview

Adrien Lastic produces BLE-controlled vibrators using the Nordic UART service with 11-byte hardcoded command packets for each mode.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Palpitation (LVS-S001) | varies | [Adrien Lastic](adrien-lastic.md) | 10 preset vibration modes |
| Palpitation Legacy (LVS-S001L) | varies | [Adrien Lastic](adrien-lastic.md) | 10 preset vibration modes |
| Revelation (LVS-S002) | varies | [Adrien Lastic](adrien-lastic.md) | 10 preset vibration modes |
| Revelation Legacy (LVS-S002L) | varies | [Adrien Lastic](adrien-lastic.md) | 10 preset vibration modes |
