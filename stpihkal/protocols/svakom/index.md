# Svakom

## Overview

Svakom is a manufacturer of BLE-controlled intimate devices. Their products use a common protocol family based on the `0xFFE0` service UUID with model-specific command structures.

## Common BLE Patterns

- Service UUID: `0000ffe0-0000-1000-8000-00805f9b34fb`
- Tx Characteristic UUID: `0000ffe1-0000-1000-8000-00805f9b34fb`
- Commands typically start with `0x55`
