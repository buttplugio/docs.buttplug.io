# Synchro

## Overview

Synchro manufactures BLE-controlled rotating devices. Devices advertise under names like "Shinkuro", "synchro2", and "synchro EX".

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Synchro | Shinkuro, synchro2 | [Synchro](synchro.md) | Rotate |
| Synchro Exchange | synchro EX | [Synchro](synchro.md) | Rotate |
