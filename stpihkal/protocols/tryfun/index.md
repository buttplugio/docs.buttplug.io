# TryFun

## Overview

TryFun produces Bluetooth LE devices with vibration, thrusting, spinning, heating, and pattern modes. The Yaun series uses a structured command protocol with a checksum byte.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Yaun Series | TRYFUN-ONE | [Protocol](tryfun.md) | Vibration, thrust, spin, heat, patterns |
