# Shockspot

## Overview

The Shockspot is a serial (RS-232/FTDI USB) connected fucking machine. It communicates via a plain-text ASCII protocol at 19200 baud.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Shockspot | N/A (serial) | [Protocol](shockspot.md) | Linear actuation (position + speed) |
