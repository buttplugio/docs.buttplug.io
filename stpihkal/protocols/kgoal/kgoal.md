# KGoal Kegelcizer Protocol

## Introduction

The KGoal Kegelcizer is a BLE pelvic floor biofeedback device. A Python implementation is available at [khole-py](https://github.com/metafetish/khole-py).

Protocol details are not yet documented here. Refer to the khole-py implementation for the full command format.

## Notes

- The `kgoal-boost.yml` device config covers the KGoal Boost variant.
- BLE advertisement name and full service/characteristic layout are not documented in the original issue.
