# Cyclone

## Overview

The Rends Cyclone X10 is a USB HID-connected rotating masturbator. A related device (臨界点, "Rinkaiten") uses the same HID protocol with vibrators instead of a rotator.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Cyclone X10 | N/A (USB HID) | [Protocol](cyclone.md) | Rotation (speed 0–10, modes 0–9) |
| 臨界点 (Rinkaiten) | N/A (USB HID) | [Protocol](cyclone.md) | Vibration (3 motors) |
