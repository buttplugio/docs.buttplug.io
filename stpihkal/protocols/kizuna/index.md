# KIZUNA

## Overview

KIZUNA produces classic Bluetooth (BT 2.x) vibrators using RFCOMM serial communication. Commands are ASCII characters terminated with `\r\n`.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| KIZUNA SMART | KIZUNA SMART | [Protocol](kizuna.md) | Vibration, modes |
