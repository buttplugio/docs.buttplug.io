# Prettylove

## Overview

Prettylove manufactures BLE-controlled vibrators. Their devices use a simple BLE protocol with a single service for control and status.

## Common BLE Patterns

- BLE name prefix: `Aogu BLE`
- Single service with tx/rx characteristics

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Aogu BLE Device | `Aogu BLE *` | [Prettylove](prettylove.md) | Vibrate |
