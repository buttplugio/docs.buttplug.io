# VibCrafter

## Overview

VibCrafter produces encrypted BLE vibrators using AES encryption and an authentication handshake before accepting text-based commands for motor, light, and heat control.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| VibCrafter | be gentle | [VibCrafter](vibcrafter.md) | Dual motor intensity, patterns, heat, light, battery |
