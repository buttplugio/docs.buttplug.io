# We-Vibe

## Overview

We-Vibe (by We-Vibe / WOW Tech Group) manufactures a range of BLE-controlled couples' vibrators and personal massagers.

## Common BLE Patterns

- Single BLE service UUID: `f000bb03-0451-4000-b000-000000000000`
- 8-byte control packets
- Device model identified via info characteristic byte 8
