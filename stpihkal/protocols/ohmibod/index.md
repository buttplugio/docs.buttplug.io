# OhMiBod

## Overview

OhMiBod (now part of Kiiroo) makes BLE-connected vibrators. The Esca 2 uses Nordic chipset firmware.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Esca 2 | OhMiBod 4.0 | [Protocol](ohmibod.md) | Vibration (0–100), light intensity |
