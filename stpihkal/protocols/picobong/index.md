# Picobong

## Overview

Picobong produces Bluetooth LE vibrators in the Remoji line. All devices share a common 3-byte command format with motor ID, mode/pattern, and speed.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Remoji Blow Hole | Picobong Male Toy | [Protocol](picobong.md) | Vibration, patterns |
| Remoji Diver | Picobong Egg | [Protocol](picobong.md) | Vibration, patterns |
| Remoji Life Guard | Picobong Ring | [Protocol](picobong.md) | Vibration, patterns |
| Remoji Surfer | Picobong Butt Plug / Egg driver / Surfer_plug | [Protocol](picobong.md) | Vibration, patterns |
