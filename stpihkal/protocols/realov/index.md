# Realov

## Overview

Realov produces BLE-controlled vibrators using a simple 4-byte command protocol for intensity control.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Realov Vibe | REALOV_VIBE | [Realov](realov.md) | Vibration intensity |
