# Motorbunny

## Overview

Motorbunny produces Bluetooth LE saddle-style vibrators and rotators. The Classic model uses an 18-byte command packet with a CRC and fixed trailer byte.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Classic | MB Controller | [Protocol](motorbunny.md) | Vibration, rotation |
