# Nobra

## Overview

Nobra is a German company that has been selling vibrators since 1999. It offers both insertable vibrators as well as a male vibrator with two motors ("Nobra Twincharger"). The digital control box can be controlled via Bluetooth using the official "NobraControl" application.
