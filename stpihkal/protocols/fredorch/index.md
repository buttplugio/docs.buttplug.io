# Fredorch

## Overview

Fredorch produces BLE-controlled sex machines including rotary and linear devices with speed control, pattern playback, and a structured packet protocol.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Fredorch Rotary (M1 series) | M1_* | [Fredorch Rotary](fredorch-rotary.md) | Speed control, rotation, patterns |
