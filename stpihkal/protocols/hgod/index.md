# Hgod

## Overview

Hgod produces Bluetooth LE vibrators sold under the "Butterfly Love" branding. The AMN NEO model uses a 6-byte command format over a proprietary service.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Butterfly Love (AMN NEO) | AMN NEO | [Protocol](hgod.md) | Vibration, patterns |
