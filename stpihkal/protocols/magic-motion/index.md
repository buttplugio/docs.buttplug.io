# Magic Motion

## Overview

Magic Motion makes a range of Bluetooth LE vibrators and couples toys. Multiple protocol versions exist across their product line; most devices share a common BLE service UUID but differ in command format.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Awaken / Equinox / Solstice (base protocol) | varies | [Protocol](magic-motion.md) | Vibration |
| Solstice X | Solstice X | [Protocol](magic-motion.md) | Dual vibration |
| Zenith | funwand | [Protocol](magic-motion.md) | Vibration |
