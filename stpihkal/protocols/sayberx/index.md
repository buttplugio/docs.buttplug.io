# SayberX

## Overview

SayberX produces interactive stroker devices consisting of a paired output device and an input ring controller (X-Ring) that communicate with a host over BLE using AT-style ASCII commands.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| SayberX Stroker | SayberX | [SayberX](sayberx.md) | Speed control (output) |
| X-Ring Controller | X-Ring 0F313B (suffix varies) | [SayberX](sayberx.md) | Speed sensor (input) |
