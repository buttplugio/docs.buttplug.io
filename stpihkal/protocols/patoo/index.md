# Patoo

## Overview

Patoo manufactures BLE-controlled devices built on the TI SimpleLink SensorTag platform. The product line includes vibrating and rotating devices with pressure sensors.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Patoo Carrot | PTVEA* | [Patoo](patoo.md) | Vibrate, Pressure sensor |
| Patoo Devil | PBT* | [Patoo](patoo.md) | Vibrate (dual motor), Pressure sensor |
| Patoo Vibrator | PCS* | [Patoo](patoo.md) | Vibrate |
| Patoo Bean Sprout | PHT* | [Patoo](patoo.md) | Vibrate |
