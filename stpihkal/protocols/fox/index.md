# Fox

## Overview

Fox produces Bluetooth LE vibrators. The protocol uses a 5-byte command for speed and pattern selection.

## Devices

| Device | BLE Name | Protocol Page | Features |
|--------|----------|---------------|----------|
| Fox (various) | FOX / FOX M70 Pro / FoxM70Pro | [Protocol](fox.md) | Vibration, patterns |
