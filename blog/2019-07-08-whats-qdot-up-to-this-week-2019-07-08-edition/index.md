### Buttplug
Anyways let's start with the boring. The library work.

<!--truncate-->

For the core libraries, I put out Buttplug C# 0.4.7 last week. Big feature there was the Xamarin module which now gives us the possibility of Android/iOS apps. There was also some toy additions and fixes (Cyclone SA should work again!) and a few development things (you can build from source zips again and don't require the full git repo). That was also released as intiface-cli-csharp 0.4.7, so if you're running Intiface Desktop on windows you probably got an update notification for that.

On the JS side, I rewrote the vue-buttplug-material-component to make life easier on all of us. The code is now much simpler, and also has a better UI that doesn't require users to know all of their IP addresses! Which leads us to...

As you probably saw from the multiple youtube/pornhub video posts...
